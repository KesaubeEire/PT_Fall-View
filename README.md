# PT_Fall-View

Mteam 专用瀑布流视图脚本.
Script only for m-team masonry view.

## packages & docs

[svelte-bricks](https://bricks.janosh.dev/)  
[Notyf](https://github.com/caroso1222/notyf)

## 功能

- 卡片常驻信息展示: 标题 / 置顶 / 免费 / 大小 / 副标题 / 标签 / 下载&收藏 / 上传时间 / 评论/上传/下载
- 卡片信息配置: 悬浮时鼠标滑动显示大图
- 卡片点击事件:
  - 点击卡片副标题进入详情页
  - 点击卡片和 iframe 按钮可以不打开新窗口查看种子详情
  - 点击卡片进入下载页
